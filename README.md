## üßê About the Project

The **Jakarta's Water Stories** project aims to provide a visual narrative of how Jakarta‚Äôs evolution‚Äîfrom the pre-colonial period to becoming a modern megacity‚Äîhas shaped its flood risks. This dashboard does not only display data but also archives local wisdom and collects citizens‚Äô stories to build collective understanding.

### ‚ú® Key Features

* **Stories Journey** ‚Äì interactive narratives on Jakarta‚Äôs flood history and its relation to urban development.
* **Timeline Journey** ‚Äì visual maps showing changes in Jakarta‚Äôs landscape from the 16th century to the present.
* **Wisdom Journey** ‚Äì an archive of local knowledge such as the situ (small lakes) system, traditional canals, and stilt architecture that remain relevant for flood control.
* **Voices Journey** ‚Äì a participatory platform where citizens can share flood experiences, mapping collective memory.
* **Album Journey** ‚Äì a visual gallery of photos, illustrations, and documents about Jakarta and water.

## üèÅ Getting Started

As this project is static, you don‚Äôt need a complicated installation. Simply clone this repository and open the main file in your browser.

### Prerequisites

You only need a modern web browser such as Chrome, Firefox, Safari, or Edge.

### Installation

1.  Clone this repository to your local computer:
    ```bash
    git clone [https://github.com/nama-anda/nama-repositori.git](https://github.com/nama-anda/nama-repositori.git)
    ```
2.  Navigate into the project directory:
    ```bash
    cd nama-repositori
    ```
3.  Open the `index.html` file in your browser. Done!
